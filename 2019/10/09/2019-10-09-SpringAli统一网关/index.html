什么是spring网关

为微服务架构提供一种简单而有效的统一的 API 路由管理方式,不仅提供统一的路由方式，并且基于 Filter 链的方式提供了网关基本的功能，例如：安全，监控/埋点，和限流等。

网关的功能特征

  1. Predicates 和 Filters 作用于特定路由
  2. 集成 Hystrix 断路器
  3. 集成 Spring Cloud DiscoveryClient
  4. 易于编写的 Predicates 和 Filters
  5. 限流和路径重写
